# vue-mall-admin

## 前言

前后端分离的商城项目，该项目为后台前端。

## 描述

该项目是使用 `Vue2.0` + `ElementUI` 开发的后台系统前端部分，主要包括以下功能：

* 用户管理
* 商品管理
* 订单管理
* 财务管理
* 仓储管理
* 活动营销
* 权限管理

## Usage

### Project setup

```bash
yarn install
```

### Compiles and hot-reloads for development

```bash
yarn run serve
```

### Compiles and minifies for production

```bash
yarn run build
```

### Run your tests

```bash
yarn run test
```

### Lints and fixes files

```bash
yarn run lint
```
